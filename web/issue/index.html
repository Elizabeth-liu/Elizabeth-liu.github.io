# vue

Vue-router: 最外层需要/，子路由不要

一些预处理程序，例如sass不能解析>>>属性，这种情况下可以用deep，它是>>>的别名，工作原理相同。

<style lang="css" scoped>
  .gHeader /deep/ .name{ //第一种写法
    color:red;
  }
  .gHeader >>> .name{   //二种写法
    color:red;
  }
</style>

css modules添加的data-v只是为了区分局部样式，跟是否能被穿透没关系

vue的props也可以设置初始值
